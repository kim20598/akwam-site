# أكوام - منصة مشاهدة الأفلام والمسلسلات

منصة عربية لمشاهدة الأفلام والمسلسلات مجاناً، مبني على سكريبر موقع أكوام الأصلي.

## المميزات

- واجهة مستخدم عربية متكاملة
- دعم الأفلام والمسلسلات بأنواعها
- دعم البحث والمفضلة
- مشغل فيديو متكامل
- نظام تخزين مؤقت للأداء
- تصميم متجاوب مع جميع الأجهزة

## تثبيت محلي

### المتطلبات

- Python 3.11+
- Redis
- Docker (اختياري)
